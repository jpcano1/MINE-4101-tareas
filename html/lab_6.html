  • Un estudio nutricional ha realizado un experimento a 78 pacientes para demostrar 3 diferentes tipos de dietas que quieren llevar al mercado. Sin embargo por falta de presupuesto solo pueden lanzar al mercado una de las dietas que experimentaron. Para ello le han propuesto realizar un estudio ANOVA para determinar cual es el tipo de dieta que deberan lanzar al mercado. Siguiendo la metodologa de ANOVA, realice los siguientes pasos:

    1. Importe los datos del archivo Diet.xlsx
    2. Realice un nuevo campo con la diferencia de peso registrado entre las 6 semanas de estudio.
    3. Verifique los supuestos y realice las modificaciones necesarias.
    4. Determine si existe suficientes pruebas estadsticas para demostrar que las dietas no son iguales.
    5. Determine el mejor tipo de dieta para lanzar al mercado

Comparación entre pares de dietas¶

Segn el anlisis de ANOVA, sabemos que las diferencias de dietas son estadsticamente significativas, pero ANOVA no dice qu dietas son significativamente diferentes entre s. Para conocer los pares de dietas diferentes significativos, realizaremos anlisis de comparacin mltiple por pares (comparacin post-hoc) usando la prueba Tukey HSD.

Supuestos de Test ANOVA¶

La prueba de Levene se puede utilizar para verificar la homogeneidad de las varianzas. Hiptesis nula: las muestras de poblaciones tienen varianzas iguales.

In [ ]:
w, pvalue = stats.levene(diet[diet['Diet']=='A']['Diff'],
                         diet[diet['Diet']=='B']['Diff'],
                         diet[diet['Diet']=='C']['Diff'])
print(w, pvalue)

0.6256665777896312 0.5376731304274011

Como el valor P no es significativo (> 0.05), no podemos rechazar la hiptesis nula de igualdad de varianzas.

La prueba de Shapiro-Wilk se puede usar para verificar la distribucin normal de los residuos. Hiptesis nula: los datos provienen de una distribucin normal.
